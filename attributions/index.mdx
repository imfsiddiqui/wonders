# Attributions

This section is dedicated to acknowledging the resources, tools, and
inspirations that have contributed to the creation and maintenance of this site.
Each item listed here has played a significant role in shaping the content,
design, and functionality of the site.

- [**Docusaurus**](https://github.com/facebook/docusaurus): A modern static
  website generator that powers this site.[\*](#docusaurus)
- [**Noto Emoji**](https://github.com/googlefonts/noto-emoji): A collection of
  emojis used throughout the site for visual appeal.[\*](#noto-emoji)
- [**Fluent Emoji**](https://github.com/microsoft/fluentui-emoji): A set of
  emojis used along with Noto Emoji for consistency and clarity in visual
  communication.[\*](#fluentui-emoji)
- [**Coloured Icons**](https://github.com/dheereshagrwal/coloured-icons): Brand
  logos and tech stack logos used in the site, providing a vibrant and
  recognizable visual identity for various technologies and
  platforms.[\*](#coloured-icons)
- [**Devicon**](https://github.com/devicons/devicon): A set of icons for
  programming languages, tools, and platforms used in the site.[\*](#devicon)
- [**Noto Sans**](https://github.com/notofonts/notofonts.github.io): A versatile
  typeface used for the site's typography.[\*](#noto-sans)
- [**JetBrains Mono**](https://github.com/JetBrains/JetBrainsMono): A monospaced
  font used for code snippets and technical content.[\*](#jetbrains-mono)
